Delete datapoints for a tag in a given time period.

If startTime and endTime are equal, then it will attempt to delete data if the passed in DataTime has data.

Parameters
tagIdThe ID of the tag to delete data from.
startTimeThe starting timestamp to delete datapoints from. (Inclusive)
endTimeThe ending timestamp in the time period. (Exclusive)
awaitCompletionIndicates whether the server should complete the delete before returning the call. If false, the call will return once the tag ID and time period have been accepted, but not necessarily once the datapoints have been deleted. If true, the call will wait for the datapoints to be deleted from disk before returning.
deadlineSecondsThe maximum number of seconds before cancelling the request. If unspecified, the default deadlineSeconds provided when constructing this client is used.
cancellationTokenA cancellation token that can be used to cancel the operation. If unspecified, no token is attached to the request.
Returns
A task that represents the asynchronous delete operation, which wraps a DeleteDataStatus.
