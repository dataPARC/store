 The slice length for reading slices of array datapoints. If greater than the length of the array, it will be truncated to avoid invalid indexes. If 0, the original array will be returned. bool UseSimpleBounds = true [get, set]
 If true then datapoint that match the timestamp requested and doesn't have quality Quality.Bad_BoundNotFound is returned. If no datapoints match the timestamps requested, then the datapoint at the previous timestamp is returned. If false and no datapoints match the timestamp requested, then a linearly interpolated datapoint is calculated from 2 bounding datapoints, if they exist. If only a begin bound is found, then a step interpolated datapoint is returned.
