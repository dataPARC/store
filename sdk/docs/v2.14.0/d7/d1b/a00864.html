enum dataPARC.TimeSeries.Core.Enums.Direction

Directions for reading datapoints.

Enumerator
Forwards 

Read datapoints forward in time, meaning each newer datapoint read has a more recent timestamp than the previous datapoint.

Backwards 

Read datapoints backward in time, meaning each newer datapoint read has a less recent timestamp than the previous datapoint.
