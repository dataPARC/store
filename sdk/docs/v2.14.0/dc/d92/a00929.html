Initializes a new instance of the ClientFactory class with an array of hostnames and port numbers.

The first server in the array of servers is set as the primary server.

Parameters
serversThe array of hostname and port number pairs.
certificateValidationThis provides methods for validating the server's certificate.
authProviderIf authentication is enabled, this is the OAuth provider for security tokens.
enableInternalLogMessagesTurn on internal communication logging used for troubleshooting.
deadlineSecondsThe default timeout for requests, in seconds. This is attached to all requests that specify a timeout, even retries to failover servers. This means a read request on the client may go longer than this value without timing out since the time restarts with each retry.
avgFailoverMinutesThe average number of minutes to wait once failover occurs before trying to revert to the primary server.
