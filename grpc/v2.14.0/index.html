ReadAtTimeParameters

Parameters for reading raw datapoints at specific timestamps.

FieldTypeLabelDescription
TagIdentifier TagQueryIdentifier

The tag to read from (required).

Timestamps google.protobuf.Timestamp repeated

The timestamps at which to read datapoints (required).

UseSimpleBounds bool

If true then the datapoint that matches the timestamp requested and doesn't have quality Quality.Bad_BoundNotFound is returned. If no datapoints match the timestamps requested, then the datapoint at the previous timestamp is returned. If false and no datapoints match the timestamp requested, then a linearly interpolated datapoint is calculated from 2 bounding datapoints, if they exist. If only a begin bound is found, then a step interpolated datapoint is returned.
